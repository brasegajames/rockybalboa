Pour utiliser notre OCR, suivre ces étapes :
- Charger une image sur laquelle travailler (Fichier/...)
- Choisir le filtre à utiliser pour la binarization (Préférences/...)
- Appliquer binarization (le nombre de fois désirés, en repassant par le choix du filtre si nécessaire), rotation, découpage des caractères, editeur (Reconnaissance/...)
- Sauvegarder le texte entré (Fichier/...)
- Quitter le programme (Fichier/...)


Raccourcis (CTRL + ... ) :
O = Charger une image
S = Sauvegarder
Q = Quitter le programme
E = Binarize
R = Rotate
T = Cut
Y = Editeur
I = Filtre
H = aide (ici)
A = A propos
